# Import Existing Files

This guide explains how to import your existing media library into Cinephage.

## Quick Overview

1. Go to **Library > Movies** or **Library > TV**
2. Click **Import** button
3. Select folder containing your media
4. Cinephage will scan and match files
5. Review matches and confirm
