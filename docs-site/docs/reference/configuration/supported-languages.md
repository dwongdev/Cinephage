# Supported Languages

This reference lists all languages supported by Cinephage for subtitles and metadata.

## Language Codes

Cinephage uses ISO 639-1 language codes:

| Language   | Code |
| ---------- | ---- |
| English    | en   |
| Spanish    | es   |
| French     | fr   |
| German     | de   |
| Italian    | it   |
| Portuguese | pt   |
| Russian    | ru   |
| Chinese    | zh   |
| Japanese   | ja   |
| Korean     | ko   |
